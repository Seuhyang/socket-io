## Intro

## scripts
1. **`npm run build`**   
   1. compile   
2. **`npm run watch`**   
   1. real time compile
3. **`npm run dev`**   
   1. pm2로 개발환경으로 서버 실행

## 주의사항
- pm2가 package 에 포함이 되지 않음.
